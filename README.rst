Xeger
=====

Library to generate random strings from regular expressions.

To install, type:

::

    pip install xeger


To use, type:

>>> from xeger import Xeger
>>> x = Xeger(limit=10)  # default limit = 10
>>> x.xeger("/json/([0-9]+)")
u'/json/15062213'
